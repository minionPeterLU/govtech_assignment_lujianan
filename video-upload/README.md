# How to run the application in video-upload terminal
- From Linux/MacOS
    - export NODE_OPTIONS=--openssl-legacy-provider
    - npm install
    - npm run start

- From Windows
    - set NODE_OPTIONS=--openssl-legacy-provider
    - npm install
    - npm run start

# environment
The program was built using ReactJS framework, es6
